1. cd ~/Desktop - переход на рабочий стол
2. Desktop> mkdir my-project - создает папку my-project на рабочем столе
3. Desktop> cd my-project - переход в паку my-project
4. my-project> pwd - показывает путь 
5. my-project> echo "Some text">file.txt - создает текстовый файл file.txt в папке my-project
6. my-project> ls - покажет содержимое папки my-project
7. my-project> cat file.txt - покажет содержимое файла file.txt 
8. my-project> rm file.txt - удалит файл file.txt
9. my-project> cd .. - переход на папку выше
10. my-project> ls -la - покажет скрытые файлы в папке my-project

------------------------------- GIT --------------------------
1) my-project> git init - инициализация git в данной папке, создается скрытую папку .git
2) git checkout - команда которая достает старую версия проекта из репозитория и помещает ее в     рабочую директорию
3) три основные команды git это git add, git commit, git checkout
4) git status - отображает текущее состояние git репозитория
5) git add <files> - подготовка файлов перед коммитом 
6) git commit -m "message" - создание коммиита с записью изменений в репозиторий
7) git log - просмотр истории изменений(коммитов)
8) git log --all - выведет все коммиты
9) git log --all --graph - выведет все коммиты в виде дерева 
10) git checkout <commit hash> - переход в определенную версию проекта по SHA1 хешу(можно ввести как полный хеш, так и начало хеша(от четырёх знаков) 
11) git checkout <branch name> - также можно перейти на другую ветку

------------------------------------Ветки в git-------------------
1) ветка это ссылка на коммит
2) git branch <branch name> создание ветки
3) git checkout <branch name> - переход в ветку
4) git checkout -b <branch name> - и создание новой ветки и переход в нее
5) git branch - отображает список всех ветвей
6) git branch -m <new branch name> - переименование текущей ветки
7) git branch -d <branch name> - удаляет ветки (текущую ветку удалить нельзя)
8) git merge <feature branch name> - слияние другой ветки(feature branch) в текущую ветку(receiving  branch)
